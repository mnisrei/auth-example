# Auth example for future use as reusable code repository 
This repo consist a basic frontend at "/client" and rest is the backend for the auth.<br />

# It consists:
1.) Auth path <br />
2.) Middleware <br />
3.) Controllers <br />
4.) A db to store the user's data
<br />
